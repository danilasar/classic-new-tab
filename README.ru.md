# Classic New Tab

[English](README.md) | Русский

Classic New Tab - расширение для Chrome, которое заменяет стандартную новую вкладку классическим экраном с приложениями, закладками и часто посещаемыми сайтами.

Что умеет расширение:

- показывает приложения Chrome и быстрые ссылки из закладок;
- показывает часто посещаемые сайты плитками;
- локально сохраняет миниатюры для видимых плиток;
- поддерживает закрепленные и пользовательские плитки;
- позволяет скрывать, возвращать, добавлять и редактировать плитки;
- содержит страницу настроек для управления плитками и очистки миниатюр.

## Установка

1. Откройте `chrome://extensions`.
2. Включите **Режим разработчика**.
3. Нажмите **Загрузить распакованное расширение**.
4. Выберите папку этого репозитория.
5. Откройте новую вкладку.

После изменения файлов вернитесь на `chrome://extensions` и перезагрузите расширение.

## Приватность

Classic New Tab хранит настройки плиток через хранилище расширений Chrome. Миниатюры снимаются со страниц, которые вы посещаете, и сохраняются локально в `chrome.storage.local`. Расширение не отправляет миниатюры или данные о посещениях на сервер.
